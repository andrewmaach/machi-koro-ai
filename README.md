# machi-koro-ai
A framework and AI to play Machi Koro, seen here: https://boardgamegeek.com/boardgame/143884/machi-koro

# Card attributes:
* title (string)
* roll (activation) (int)
* build-cost (int)
* kind (restaurant, grain, )
* rule ()
* requires-harbor (boolean)
* steal
* landmark-requirement [boolean]
* landmark-limit [boolean]
